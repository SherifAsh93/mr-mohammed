# Hidden Admin Access

The admin dashboard should not have a visible link in the public website.

Admin access flow:

1. User clicks the main website logo 3 times.

2. A password input appears.

3. Enter the admin password.

Initial password:

123456

4. If the password is correct, open the admin dashboard.

The admin password must be stored in a way that allows changing it easily.

Inside the admin dashboard, provide a settings section where the teacher can:

- Change admin password
- Confirm the new password
- Save the new password

After changing the password, the old password should no longer work.

The purpose of this system is to allow the website owner to receive the project
with a default password and then change it after delivery.
